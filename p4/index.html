P4 Demo App

About

This application is for demonstrating the functionalities and features in P4. It contains examples to show the usage of P4's declarative grammar and syntax for specifying different data transformations, visualizations, and interactions.


System Requirements

It is best to run this application using the latest version of Google Chrome on a system with a modern GPU. Please note that P4 is still in beta. Both Chrome and Firefox with AMD, Intel, and Nvidia GPUs have been tested on Linux, Mac OS, and Windows. It is known that Mac OS with some AMD GPUs can cause error.


Dataset

The dataset used in this demo application contains data records of babies born in 2015 and their parents, which is part of the publicly-available datasets from U.S. National Center for Health Statistics released to public.
